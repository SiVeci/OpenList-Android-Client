# v0.2 Backlog

v0.1 的架构目标是"新增上传、分享、预览、任务中心时不需要重写网络层、实例层、Token 层、数据库主键设计"（[v0.1_EXECUTION_PLAN.md](v0.1_EXECUTION_PLAN.md) §15）。以下条目均已有对应的包/接口/DAO 预留位，是在预留骨架上"填肉"，而非从零设计。

## v0.2：上传与文件操作

- `feature/upload`：从系统文件选择器多选上传，后台任务 + 通知进度。
- `FileOperationRepository`：新建目录、重命名、批量重命名、移动、复制、删除。
- `UploadTaskEntity` + WorkManager 上传任务、前台服务通知。
- 文件列表页补批量选择（长按进入多选模式）。

## v0.2+：下载器增强（KNOWN_ISSUES.md 呼应项）

- 监听 `DownloadManager.ACTION_DOWNLOAD_COMPLETE`，把 `DownloadTaskEntity.status` 真正更新为 SUCCESSFUL/FAILED。
- 应用内下载历史/任务列表页。
- 断点续传、并发下载、暂停/继续/重试、大文件校验（自研 OkHttp 流式下载器，逐步替代 DownloadManager）。

## v0.3：分享与任务中心

- `feature/share` + `ShareRepository` + `ShareEntity`：创建/查看/启停/删除分享、复制分享链接。
- `feature/task` + `TaskRepository` + `RemoteTaskEntity`：后端任务轮询、离线下载入口。

## v0.4：预览与播放器

- `feature/preview` + `PreviewRepository` + `PreviewCacheEntity`：图片/文本/Markdown 预览。
- Media3 播放器封装（视频/音频）、Coil 图片加载、字幕入口。
- 文件详情页补预览入口。

## v0.5：轻量管理台

- `feature/admin` + `AdminRepository` + `AdminPermissionChecker`：存储/用户/任务列表，API Token 重置。
- `feature/webfallback`：复杂管理表单/SSO/WebAuthn 的 WebView/Custom Tabs 兜底。

## 鉴权增强（散落在 v0.2+ 各阶段，优先级按需排）

- 2FA OTP 输入、LDAP 登录入口、SSO（Custom Tabs）、原生 WebAuthn/Passkey。
- Hash 登录（`/api/auth/login/hash`，客户端预哈希，避免明文密码上线）。
- 自签证书支持（用户主动开启 + 明确风险提示）。
